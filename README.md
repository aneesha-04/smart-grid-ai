# Intelligent Grid AI ⚡
AI-powered renewable energy monitoring and prediction system for smart grids.
## Features
- Real-time solar/wind energy monitoring  
- Energy demand prediction  
- Fault & overload alerts  
- Smart power distribution  
## Tech Stack
HTML, CSS, JavaScript, Python, Node.js, MongoDB
## Use Cases
Smart cities, solar farms, utility companies.
## Future Scope
IoT integration, better forecasting, and mobile app support.
